# Intelligent Traffic Monitoring System

Final year project — Dept. of CSE-AI, NIET  
Team: Vaibhav & Sahil Dagar

## Modules
- Vehicle detection (YOLOv8)
- Speed estimation (homography)
- License plate OCR (EasyOCR)
- Emergency vehicle detection
- Accident detection (CNN-LSTM)
- Real-time alerts (Twilio)

## Setup
pip install -r requirements.txt
